# Video-transcription-and-Celebrity-Identification
Video transcription and Celebrity Identification

Celebrity Recognition + Audio Transcription (Gemini 1.5)

A Streamlit application that lets you upload any video, automatically detect celebrities in the frames, and extract spoken text from the audio using Google's Gemini 1.5 Flash/Pro models.

This project combines Computer Vision, Face Recognition, and Generative AI into an easy-to-use interface.
